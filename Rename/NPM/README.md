# NPM (Node Package Manager)

## 정의
Node.js로 만들어진 모듈을 웹에서 받아서 설치하고 관리해주는 프로그램입니다.

## 방법
사용하기 위해선 사전에 npm의 설치가 필요하다. 예전에는 npm의 추가 설치를 요하였지만 지금은 node.js를 설치하면 내장되어 있다고 한다.

아래는 대략적인 명령어를 정리해봤다.

```
// 메인 설명
npm help

// 명령어에 대한 설명
npm <명령어> -h

// npm에서 관리되는 모듈 리스트
npm list

// 모듈의 버전별 상세 정보
npm view <패키지명>

// 설치되어 있는 모듈의 목록 리스트
npm list installed

// 설치된 패키지를 최신 버전으로 업데이트
npm update <패키지명>

// 모듈을 찾아 설치
npm install <패키지명>
```
